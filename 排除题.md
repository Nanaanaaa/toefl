# 排除题

## 特征识别: EXCEPT / NOT

## 做题顺序: 审定顺归
1) S: EXCEPT/NOT
2) D: 四个选项选出>=2组定位词
   原则: 定位词中的名词 不与文章标题, 题干, 选项间重复
3) S顺序 + G改写: 按照原文句子的顺序分别对应选项, 同时去判断正误, 选的是**错误**的一项.

## 错误原因:
1. 无
2. 误(与原文表达不符)
3. 偏(偏题, 不符合题目要求)
4. 归(归因)

## notice:
1. 先看选项找出每个选项所对应的关键词
2. 然后扫描段落, 观察到关键词属于哪个选项, 做标记
3. 不同主语但是出现了相同的选项内容则这个选项要选这个选项
4. 选项选择关键词 这些选项选择的关键词和文章标题, 题目主语 交集一定为空
